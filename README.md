# Pyorekt

Pyorekt is an open-source self-hosted Git repository management system written in C/C++ and Vue.js. It provides a lightweight and efficient way to manage Git repositories on your own server. Pyorekt use dual licensing: a free Community edition and a paid Pro edition with additional features.

## Community Features

- Integration with CI/CD pipelines
